WhatsApp

WhatsApp is a cross-messaging platform founded in 2009, created by Brian Acton and Jan Koum that was created by and later bought by Facebook

Uses for college

For anyone wondering how such a social media tool such as WhatsApp could be used for college students and lecturers here is how:

  1. Using WhatsApp allows the user to message and call othere people.
  2. Group chats are available through this app and allows for quick input if need be by either a student or lecturer.
